# Mentes Creativas - Ciencias Naturales

Aplicacion educativa multimedia para estudiantes de cuarto y quinto grado del Colegio Mentes Creativas.

## Temas incluidos

- Ecosistemas.
- Ciclo del agua.
- Cuerpo humano.

## Stack tecnico

- React 19.
- Vite 7.
- TypeScript.
- Jest 30.
- React Testing Library.
- GitHub Actions.
- Vercel.

Requisito de Node.js: `>=20.19.0`. Para GitHub Actions se usa Node.js 22.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Pruebas y build

```bash
npm test
npm run build
```

## Despliegue en Vercel

URL publicada:

```text
https://mentes-creativas-ciencias-naturales-rho.vercel.app
```

Configuracion recomendada en Vercel:

- Framework preset: `Vite`.
- Install command: `npm install`.
- Build command: `npm run build`.
- Output directory: `dist`.
